# Oasis — Decentralized Social Platform with Tokenized Governance

## Overview

Built a decentralized social platform prototype using DAO governance, smart-contract execution, decentralized storage, and token-based incentives to give users more ownership over content and community moderation.

Designed the system architecture for a 48-hour hackathon, connecting a React frontend to a blockchain execution layer with governance, rewards, staking, and decentralized content storage.

**Impact: 1st place out of 100+ teams, demonstrating the ability to translate abstract Web3 product requirements into an end-to-end system architecture under severe time constraints.**

## Key Contributions

- Designed a **decentralized social platform architecture** combining frontend interaction, smart-contract execution, DAO governance, and decentralized storage  
- Built the system around a **React frontend** connected to blockchain-based posting, voting, and reward workflows  
- Modeled an **execution layer** containing governance logic, incentive mechanisms, and staking/reward pool functionality  
- Designed a **DAO-based moderation system** where users participate in content validation through voting behavior  
- Incorporated **token-based incentives** to reward users for meaningful engagement and community participation  
- Structured the platform around a **staking and rewards pool**, aligning user activity with governance and economic outcomes  
- Used decentralized storage concepts such as **Crust / IPFS** to support persistent content storage  
- Developed a clear architecture diagram showing the flow from user wallet → frontend → smart contracts → blockchain → decentralized storage

## Business Impact

- Won **1st place out of 100+ teams** in a 48-hour hackathon  
- Demonstrated how decentralized infrastructure can support social platforms with community-owned governance  
- Translated problems in traditional social media, including centralized control and misaligned incentives, into a programmable governance architecture

## Tech Stack

- **Frontend:** React  
- **Blockchain:** Moonbeam / Polkadot  
- **Execution Layer:** Smart contracts, governance module, incentive engine, staking / rewards pool  
- **Storage:** Crust, IPFS  
- **Architecture:** User wallet → frontend → execution layer → blockchain → decentralized storage  
- **Focus Areas:** DAO governance, token incentives, decentralized social platforms, smart-contract workflows, Web3 system architecture
